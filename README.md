# NEXUS

NEXUS is a premium AI Developer Command Center for Adnan Ahmed Bhameshan. This repository is currently scoped to Phase 1: product architecture, design system, routing, layout, reusable UI primitives, and responsive foundation.

## Phase 1 Scope

- React, TypeScript, Vite, Tailwind CSS
- Framer Motion, React Router, Zustand, TanStack Query
- NEXUS dark design system and theme tokens
- Boot sequence with skip preference
- Dashboard shell with sidebar, topbar, mobile navigation
- MVP route placeholders and project data foundation
- Backend/API/database architecture documentation and FastAPI skeleton

## Commands

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

The frontend dev server runs on `http://localhost:5173`.
